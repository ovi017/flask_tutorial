রাউটিং

মডার্ণ ওয়েব ফ্রেমওয়ার্কগুলো রাউটিং টেকনিক ব্যবহার করে থাকে। সরাসরি ওয়েবের কোনো পেজ অ্যাক্সেস করতে এটা খবই হেল্পফুল।

আমরা আগের চ্যাপ্টারেই তা ব্যবহার করেছি। হ্যাঁ, ওই route() ডেকোরেটরটা।

route(rule, options):

  • rule প্যারামিটারটা ব্যবহার করা হয় কোন URL এ আমরা আমাদের আউটপুট দেখাতে চাই।

  • options প্যারামিটার এখন জানার দরাকার নেই।

from flask import Flask
app = Flask(__name__)

@app.route("/love")
def patriot():
    return "I love my country"

if __name__ == '__main__': 
   app.run()

উপরের উদাহরণে আমরা "/love" রুলটা patriot() ফাংশনের সাথে বাউন্ড করা হয়েছে। এখন যদি আমরা http://localhost:5000/love URL ভিজিট করে তাহলে patriot() ফাংশনের আউটপুটটা ব্রাউজারে রিন্ডার হবে।

আমরা add_rule_url() ফাংশনও ব্যবহার করতে পারি একই কাজের জন্য। এর জন্য আগে আমাদের কাঙ্ক্ষিত ফাংশনটা লিখে নিতে হবে, তারপর উক্ত ফাংশনটা ব্যবহার করে আমাদের ফাংশনটা বাউন্ড করে নিতে পারি।

def patriot():
    return "I love my country"
app.add_url_rule(‘/’, ‘love’, patriot)
